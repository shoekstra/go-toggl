# go-toggl

A modern Go client library for the Toggl Track API v9.

## Development

### Prerequisites

- Go 1.21+
- Task (<https://taskfile.dev>)
- golangci-lint

### Common Tasks

```bash
# Run all tests
task test

# Format code
task fmt

# Run linter
task lint

# Run everything: fmt, vet, lint, test
task all

# View coverage
task test:coverage

# Clean build artifacts
task clean
```
