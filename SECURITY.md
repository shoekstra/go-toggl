# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it privately via [GitHub's security advisory feature](https://github.com/shoekstra/go-toggl/security/advisories/new) rather than opening a public issue.

Please include as much detail as possible:

- A description of the vulnerability and its potential impact
- Steps to reproduce the issue
- Any suggested mitigations

You can expect an acknowledgement within 48 hours and a resolution or status update within 14 days.
